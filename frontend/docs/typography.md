# Typography

Our typography system provides a consistent hierarchy and improves readability across all interfaces.

## Font Families

### Primary Font Stack

- **System Fonts**: Uses system fonts for optimal performance
- **Fallbacks**: Includes appropriate fallback fonts

## Type Scale

Our type scale follows a consistent ratio for harmonious sizing:

- **Display**: 48px / 3rem
- **H1**: 36px / 2.25rem
- **H2**: 30px / 1.875rem
- **H3**: 24px / 1.5rem
- **H4**: 20px / 1.25rem
- **Body**: 16px / 1rem
- **Small**: 14px / 0.875rem
- **XSmall**: 12px / 0.75rem

## Font Weights

- **Light**: 300
- **Regular**: 400
- **Medium**: 500
- **Semibold**: 600
- **Bold**: 700

## Usage Guidelines

### Headings

- Use headings to establish hierarchy
- Don't skip heading levels
- Keep headings concise and descriptive

### Body Text

- Use 16px as the base font size
- Maintain line height of 1.5 for readability
- Use appropriate font weights for emphasis

### Links

- Use indigo-600 for link colors
- Provide clear hover states
- Ensure sufficient contrast

## Examples

```html
<h1>Page Title</h1>
<h2>Section Heading</h2>
<p>Body text with proper line height and spacing.</p>
<a href="#">Link text</a>
```
