# Buttons

Buttons are interactive elements that trigger actions.

## Button Types

### Primary

Use for the main action on a page or in a section.

### Secondary

Use for supporting actions that are less prominent.

### Tertiary

Use for subtle actions or when space is limited.

## States

- **Default**: Normal state
- **Hover**: When user hovers over button
- **Active**: When button is pressed
- **Disabled**: When action is not available
- **Loading**: When action is in progress

## Usage Guidelines

- Use clear, action-oriented labels
- Place primary actions prominently
- Limit the number of primary buttons per page
- Provide appropriate feedback for all states

## Examples

```html
<button class="btn-primary">Primary Action</button>
<button class="btn-secondary">Secondary Action</button>
<button class="btn-tertiary">Tertiary Action</button>
```
