# Integration Troubleshooting

Common integration issues and solutions.

## Argos Integration

### Connection Failed

**Symptoms**: Cannot connect to Argos

**Solutions**:
1. Verify credentials (project ID, token)
2. Check base URL is correct
3. Verify network connectivity
4. Check Argos service status

### Screenshots Not Uploading

**Symptoms**: Screenshots not appearing in Argos

**Solutions**:
1. Check file paths
2. Verify Argos CLI is installed
3. Check network connectivity
4. Review Argos logs

## OpenReplay Integration

### Sessions Not Recording

**Symptoms**: No sessions in OpenReplay

**Solutions**:
1. Verify integration code is added
2. Check project key is correct
3. Verify network connectivity
4. Check browser console for errors

### Missing Session Data

**Symptoms**: Incomplete session data

**Solutions**:
1. Check privacy settings
2. Verify capture configuration
3. Review API key permissions
4. Check OpenReplay logs
